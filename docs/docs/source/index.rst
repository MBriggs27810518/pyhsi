Welcome to Human-structure Interaction: An open-source python toolbox's documentation!
======================================================================================
**pyhsi** is a python toolbox which provides *user-friendly* and *extensible* implementation of the **human-structure
interaction** simulation. The toolbox serves as a software tool that allows users to simulate and analyze the interaction
between humans and structures, particularly in the vertical direction.

.. note::

   This project is under active development.
