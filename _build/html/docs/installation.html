Installation¶

Stable release¶

To install Test Python Package, run this command in your terminal:

$ pip install test_python_package

This is the preferred method to install Test Python Package, as it will always install the most recent stable release.

If you don’t have pip installed, this Python installation guide can guide you through the process.

From sources¶

The sources for Test Python Package can be downloaded from the Github repo.

You can either clone the public repository:

$ git clone git://github.com/J-Dau/test_python_package

Or download the tarball:

$ curl -OJL https://github.com/J-Dau/test_python_package/tarball/master

Once you have a copy of the source, you can install it with:

$ python setup.py install
